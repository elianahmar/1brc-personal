# 1BRC Go

My implementation of the 1BRC Challenge.

Best recorded time: **3.88 seconds** using the solution from `p17.go`.

Here is the link to get started with the challenge

https://github.com/gunnarmorling/1brc#running-the-challenge

After you have the measurements.txt file you can my implementation using the following command

```bash
go run main.go p17 <path_to_measurements_file>
```
